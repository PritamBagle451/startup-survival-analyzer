## Startup Survival Analyzer

A fun and interactive React web application that predicts whether a fictional startup can survive, become a unicorn, pivot to AI, or completely collapse.

## Tech Stack

* React
* CSS
* JavaScript
* Recharts
* Vite

## Features

* Dynamic startup survival analysis
* Unicorn probability calculation
* AI pivot prediction
* Burn rate and runway analysis
* Interactive pie chart visualization
* Responsive UI
* Animated transitions and hover effects
* Random startup consultant insights
* Modern glassmorphism UI

## Deployment

Deployed using Vercel
